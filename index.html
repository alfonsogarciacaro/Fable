F# |> BABEL

The compiler that emits JavaScript you can be proud of!

Fable brings together the power of the F# compiler and Babel to make JavaScript a true backend for F#. Some of its main features are:

Works directly on F# source code, no compilation to .NET bytecode needed Optimizes F# code to generate as clean JavaScript as possible Passes location data to Babel to generate source maps Compatible with all Babel plugins and other JS development tools, like Webpack Support for most of the F# core library and a bit of .NET Base Class Library Tiny core library included (less than 10KB minified and gzipped) with no runtime Organizes code using ES6 modules Interacts seamlessly with other JavaScript libraries Bonus: compile NUnit tests to Mocha
